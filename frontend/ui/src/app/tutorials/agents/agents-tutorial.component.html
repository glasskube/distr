Applications and Agents

In this tutorial you will learn how to set up a Distr agent on your own machine and deploy a docker compose app with it.

We have prepared a sample app called hello-distr, that will be created in your Distr organization as soon as you start this tutorial.

  1. Deploy hello-distr

    Make sure to have docker installed on the target machine.

    If you are on Mac, please make sure to check this guide first!

  2. Install Agent

    In your terminal, execute the setup command to start the Distr agent and have it connect to the Distr Hub.
