Artifact Registry

In this tutorial you will learn how to make use of the Artifact Registry.

  1. Prepare Registry

    Your organization needs to have a slug in order to use the registry. You only need to do this once for your organization.

  2. Set Organization Slug

    The slug will be used as part of the url for artifact registry. It must start with a lower case letter or digit and consist only of lowercase letters, digits, as well as the following special characters: “.”, “_”, “__”, “-”.
