Applications

An application is the software you distribute to customers. We currently support distributing applications via docker-compose files with Helm charts available soon. To offer different channels (e.g., staging and production), you can create separate applications.

Learn more

Customers

You can invite customers to install your applications via email, providing a link to the distr.sh Customer Portal with installation instructions. You can track their installation status, version, and configuration options.
Currently, only one email address can be assigned per customer.

Learn more

Deployments

Deployments represent application installations. You can create deployments yourself if you manage the installation. As a vendor, you also have an overview of all deployments managed by your or customers or other users of your organization.

Learn more
